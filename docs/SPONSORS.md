# Sponsor integrations

| Sponsor        | Where it plugs into pipeline       | What it proves                                        | How you will demo it                                   |
| -------------- | ---------------------------------- | ----------------------------------------------------- | ------------------------------------------------------ |
| Token Company  | Compression of transfer packet     | Long clinical text can be safely compressed at scale  | Show token counts before and after compression        |
| LeanMCP        | MCP deployment and orchestration   | NeuroCast can run as a modular MCP control layer      | Call the NeuroCast MCP from the web app and voice bot |
| Wood Wide      | Numeric KPIs and metrics           | Deterministic scoring on top of LLM-extracted fields  | Visualize KPIs and which thresholds triggered routing  |
| LiveKit        | Real-time voice agent              | Clinicians can ask safe questions hands-free          | Run the Voice Commander page with a live call         |
| Phoenix        | Traces and evals                   | End-to-end observability and evaluation of the system | Show traces, latencies, and eval dashboards           |
| Kairo (option) | Audit hash ledger for key events   | Tamper-evident record of pipeline decisions           | Display a hash trail for key routing decisions        |
